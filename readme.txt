## This is a Flutter application designed for real-time image classification of monuments of Patan Durbar Square using the device's camera. The app captures images through the camera and classifies them using a TensorFlow Lite model.

Features
Real-time image classification through the device camera.
Displays classification results with confidence levels.
User-friendly UI that integrates with the device camera.
Flutter framework powered by TensorFlow Lite for efficient on-device classification.

Tech Stack
Flutter: Cross-platform mobile framework.
TensorFlow Lite: Lightweight model for on-device machine learning.
Dart: Programming language for Flutter.
Camera Plugin: Captures real-time images.
TFLite Flutter Plugin: Integrates TensorFlow Lite with Flutter.

Prerequisites
Android Studio or Visual Studio Code (IDE of choice)
Flutter SDK: Install Flutter
A device with camera support or an emulator
